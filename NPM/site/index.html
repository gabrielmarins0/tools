Instalando

Baixe o Node.js clicando aqui e faça a instalação. Junto será instalado o NPM.

Confira no terminal se o node.js e o npm foram instalados corretamente com os comandos node -v e npm -v


Npm init e package.json

Na pasta do projeto, use npm init para criar o package.json. Nele conterão as informações do projeto, use a opção -y para criar o arquivo diretamente, sem responder as perguntas.

Use npm run "nome do scrip" para executar o scripts declarados no package.json na chave scripts

Nunca alterar os arquivos .lock pois sao gerados automaticamente e contem informações importantes para o funcionamento do npm

Previamente configure o arquivo .gitignore com node_modules para ignorar esta pasta nos commits.


Instalando e removendo pacotes

Use npm install "nome do pacote" para instalar um pacote especifico. Também pode ser usado o alias npm i "nome do pacote"

Use npm uninstall | remove "nome do pacote" para remover um pacote especifico.

Use npm install para instalar ou remover pacotes, com base no arquivo package.json, chave dependencies :
Pacotes não presentes em node_modules e declarados serão instalados.
Pacotes presentes em node_modules e não declarados serão removidos.

Use --save-dev ou -D para instalar um pacote de desenvolvimento. Ele ficará declarado na chave devDependencies , separado dos demais pacotes da chave dependencies . Visualmente na pasta node_modules não existe uma separação.


Extras: Yarn

Instale conforme orientações do link https://yarnpkg.com/getting-started/install

Use yarn init para para criar o package.json

Use yarn add "nome do pacote" para instalar o pacote especificado.

Use yarn remove "nome do pacote" para remover o pacote especificado.

Use yarn upgrade "nome do pacote" para atualizar pacote especificado.

Use yarn upgrade para atualizar todos os pacotes

Use yarn como alias para instalar e remover os pacotes conforme as dependencies declaradas no arquivo package.json

Usar apenas um do gerenciadores em um projeto, ou npm, ou yarn, para evitar conflitos de configuração.
